# ьЃђВъЁ ВХћвАа(Type Inference)

ьЃђВъЁ ВХћвАаВЮ┤въђ ьЃђВъЁВіцьЂгвдйьіИЖ░ђ ВйћвЊювЦ╝ ьЋ┤ВёЮьЋ┤ вѓўЖ░ђвіћ вЈЎВъЉВЮё ВЮўв»ИьЋЕвІѕвІц.

# ьЃђВъЁ ВХћвАаВЮў ЖИ░в│И

ьЃђВъЁВіцьЂгвдйьіИЖ░ђ ьЃђВъЁ ВХћвАаВЮё ьЋ┤вѓўЖ░ђвіћ Ж│╝ВаЋВЮђ вІцВЮїЖ│╝ Ж░ЎВіхвІѕвІц.

ВюёВЎђ Ж░ЎВЮ┤ xВЌљ вїђьЋю ьЃђВъЁВЮё вћ░вАю ВДђВаЋьЋўВДђ ВЋівЇћвЮ╝вЈё ВЮ╝вІе xвіћ numberвАю Ж░ёВБ╝вљЕвІѕвІц. ВЮ┤ваЄЖ▓ї в│ђВѕўвЦ╝ ВёаВќИьЋўЖ▒░вѓў В┤ѕЖИ░ьЎћ ьЋа вЋї ьЃђВъЁВЮ┤ ВХћвАављЕвІѕвІц. ВЮ┤ВЎИВЌљвЈё в│ђВѕў, ВєЇВё▒, ВЮИВъљВЮў ЖИ░в│И Ж░њ, ьЋеВѕўВЮў в░ўьЎў Ж░њ вЊ▒ВЮё ВёцВаЋьЋа вЋї ьЃђВъЁ ВХћвАаВЮ┤ ВЮ╝Вќ┤вѓЕвІѕвІц.

# Ж░ђВъЦ ВаЂВаѕьЋю ьЃђВъЁ(Best Common Type)

ьЃђВъЁВЮђ в│┤ьєх вфЄ Ж░юВЮў ьЉюьўёВІЮ(ВйћвЊю)ВЮё в░ћьЃЋВю╝вАю ьЃђВъЁВЮё ВХћвАаьЋЕвІѕвІц. ЖиИвдгЖ│а ЖиИ ьЉюьўёВІЮВЮё ВЮ┤ВџЕьЋўВЌг Ж░ђВъЦ Жи╝ВаЉьЋю ьЃђВъЁВЮё ВХћвАаьЋўЖ▓ї вљўвіћвЇ░ ВЮ┤ Ж░ђВъЦ Жи╝ВаЉьЋю ьЃђВъЁВЮё Best Common TypeВЮ┤вЮ╝Ж│а ьЋЕвІѕвІц.

ВъаЖ╣љ ВўѕВаювЦ╝ в│┤Ж▓аВіхвІѕвІц.

Вюё в│ђВѕў arrВЮў ьЃђВъЁВЮё ВХћвАаьЋўЖИ░ ВюёьЋ┤Вёювіћ в░░ВЌ┤ВЮў Ж░Ђ ВЋёВЮ┤ьЁюВЮё Вѓ┤ьј┤в┤љВЋ╝ ьЋЕвІѕвІц. в░░ВЌ┤ВЮў Ж░Ђ ВЋёВЮ┤ьЁюВЮў ьЃђВъЁВЮђ ьЂгЖ▓ї numberВЎђ nullвАю ЖхгвХёвљЕвІѕвІц. ВЮ┤ вЋї Best Common Type ВЋїЖ│авдгВдўВю╝вАю вІцвЦИ ьЃђВъЁвЊцЖ│╝ Ж░ђВъЦ Въў ьўИьЎўвљўвіћ ьЃђВъЁВЮё ВёаВаЋьЋЕвІѕвІц.

# вгИвДЦВЃЂВЮў ьЃђВЮ┤ьЋЉ(Contextual Typing)
